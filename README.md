## Quick install
Clone this repository to any location and then install it.

Cloning
```bash
$ git clone https://github.com/nmih/ssbio.git
```

Installation
```bash
$ cd ssbio
$ python setup.py develop
```

Updating
```bash
$ cd ssbio
$ git pull
```
